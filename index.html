☁️ Sincronizare Google Drive
⚪ Nu este configurat
Datele rămân doar pe telefon. Configurează Google Drive pentru backup automat și sincronizare între device-uri.
Cum obțin Client ID? ▾
Pașii pentru Google OAuth Client ID:
  1. Deschide console.cloud.google.com
  2. Create Project → nume liber (ex: "Fitness Jurnal")
  3. În meniu: APIs & Services → Enabled APIs → Enable "Google Drive API"
  4. APIs & Services → OAuth consent screen → External → completează doar numele app-ului și email-ul tău → Save
  5. La pasul "Audience/Test users" adaugă email-ul tău
  6. APIs & Services → Credentials → Create Credentials → OAuth client ID → Web application
  7. La "Authorized JavaScript origins" adaugă: https://crstefan-lab.github.io
  8. Create → copiază Client ID-ul (format: ...apps.googleusercontent.com)
  9. Deschide index.html, la începutul scriptului pune:
    const GOOGLE_CLIENT_ID = 'TOKEN_AICI.apps.googleusercontent.com';
  10. Push pe GitHub, reîncarcă app-ul, apasă "Conectează Drive"
